```<canvas>``` — это тег, предназначенный для отрисовки графики в браузере. Чтобы с ним начать работу, конечно же, нужно добавить его в разметку.

Ок, тег в разметке есть, теперь давайте перейдём к JavaScript и начнём работу с канвасом. Для начала нужно получить доступ к самому тегу ```<canvas>```, например, с помощью метода ```getElementById```.

Канвас — это просто тег, этакий контейнер. Чтобы что-то нарисовать на нем, нужно получить объект, который занимается непосредственно отрисовкой. Такой объект называется «контекстом отрисовки».

Существует несколько видов контекстов отрисовки, а самый распространённый — ```2d```. С ним-то мы и будем работать. Чтобы получить объект контекста, нужно воспользоваться методом ```getContext``` у объекта ```canvas```.

Обычно, переменную в которой записан контекст, называют ```ctx```.

Сама отрисовка на канвасе осуществляется вызовом методов контекста, а параметры отрисовки задаются свойствами.

Например, чтобы отрисовать на канвасе закрашенный прямоугольник, нужно вызвать метод контекста ```fillRect``` и передать в него необходимые аргументы. Этот метод ожидает четыре аргумента, первые два задают координаты левого верхнего угла, третий и четвёртый — ширину и высоту прямоугольника.

Чтобы задать стиль заливки прямоугольника, перед отрисовкой прямоугольника нужно задать свойство ```fillStyle```. Цвет заливки задается так же, как это делается в CSS — с помощью названия цвета или в любом цветовом формате: HEX-код, rgb, rgba, hsl и так далее.

Последний заданный ```fillStyle``` будет применён ко всем вызовам методов ```fill*```. Чтобы залить следующую фигуру другим цветом, нужно переопределить ```fillStyle```

Аналогично свойству ```fillStyle```, с помощью ```strokeStyle``` можно задавать стиль обводки фигур, которые мы собираемся нарисовать.

Для очистки холста, используется метод ``clearRect``.
